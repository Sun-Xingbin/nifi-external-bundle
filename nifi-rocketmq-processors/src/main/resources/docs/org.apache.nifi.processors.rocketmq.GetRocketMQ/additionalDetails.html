Description:

This Processors polls Rocket MQ for data. When a message is received from RocketMQ, this Processor emits a FlowFile where the content of the FlowFile is the value of the RocketMQ message. If the message has a key associated with it, an attribute named rocketmq.key will be added to the FlowFile, with the value being the UTF-8 Encoded value of the Message's Key.
